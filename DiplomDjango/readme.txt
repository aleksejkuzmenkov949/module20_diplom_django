Проект DiplomDjango представляет собой веб-приложение на основе фреймворка Django, предназначенное для управления заметками через приложение DjangoNotes.
Оно предлагает интуитивно понятный интерфейс для создания, редактирования и удаления заметок, используя архитектуру MVT (Model-View-Template).
Модели обрабатывают данные, представления управляют логикой, а шаблоны отображают информацию пользователю.
Приложение активно использует Django Forms для упрощения процесса создания и валидации форм.
Пользовательские функции регистрации и входа в систему повышают надежность и удобство взаимодействия, предлагая эффективный инструмент для организации заметок.

Чтобы запустить проект в pycharm нужно прописать в терминале коанду :python manage.py runserver

Требования к проекту указаны в requirements.txt

При переходе по ссылке  http://127.0.0.1:8000/ запускается  проект.

Дальнейшие внутренние действия в проекте отображены в приложенных скриншотах.
